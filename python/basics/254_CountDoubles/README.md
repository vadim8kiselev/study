#254. Количество двойников

Задана последовательность целых чисел. Найдите количество таких ее элементов x, что значение 2*x содержится в последовательности.

##Входные данные
В первой строке записано число N (1 <= 100 <= N) - количество элементов последовательности. Далее записаны сами элементы, каждый в отдельной строке. Каждый элемент не превосходит по абсолютной величине 100.

##Выходные данные
Выведите количество таких элементов x, что значение 2*x содержится в последовательности.
